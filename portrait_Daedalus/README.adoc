== 1.説明
https://github.com/Xen-Shuto/species_portrait_template[species_portrait_template] で作成したオリジナルポートレートです。 +
機械種族のポートレート「機巧天使」が追加されます。 +
名称リスト「機巧天使」も追加されます。 +
女性の画像のみですので、名前設定時に性別を女性または性別不詳を選択してください。 +
名称リストは自由に選択してください。 +
 +
名称リストは個別的機械 2ベースとなっています。 +
 +
Steamワークショップでも公開しています。 +
https://steamcommunity.com/sharedfiles/filedetails/?id=3647504504 +
